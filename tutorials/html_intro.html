What actually is a website?

A website is just a very simple document (a lot like Word), formatted in a way that your web browser (chrome, firefox, etc.) understands.
It's mostly written in the Hypertext Markup Language (Thanks, 90s!), otherwise known as HTML.
There's also styling - Fonts, colours, etc. - written in CSS (Cascading Style Sheets), and some scripting, usually with the dumpster fire of a language we call JavaScript (not to be confused with Java, they're totally different.)

All of those little things between angle brackets <> are HTML tags, and that's how we format websites. These are the ones you just saw - match them with the picture above and you'll get a good idea how this works:

  • <html>: This is a HTML document - Good to know
  • <head>: This is the bit that your browser and google look at. <title>, for example, is what appears in the tab bar at the top of your browser.
    Ignore the <link> for now.
  • <h1>: This is a Heading - Level 1 is the biggest, and it goes down to <h6>
  • <p>: This is a paragraph - Just some basic text
  • <br>: This is a line break - It gives a new line without starting a new paragraph.
  • <b>: This is bold.

How do I write my own?

Getting the tools

To get started you'll need 2 programs - Something to write your HTML, and something else to look at it. You've already got a web browser to display websites, go you! We just need a plain text editor.
Word is not a plain text editor. It uses its own format which is full of rich formatting, pictures, fonts and whatever else that we really don't want.
Notepad is a plain text editor, though not a very good one. It will work, but you won't have a great time writing in it.
I'm currently writing this in (neo)Vim - It's nearly 25 years old, barely respects the mouse, has no menus, is a pain to learn, and I love it. But I wouldn't recommend it to start.
Instead, I'd look at Microsoft't Visual Studio Code. It has lots of great features, is a professional tool, and it's pretty friendly to beginners.

Creating a website
